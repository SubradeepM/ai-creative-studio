## 🤖 AI-Powered Image Style Transfer

Transform your photos into stunning artwork using artificial intelligence! This app runs entirely in your browser - no server, no API keys, no privacy concerns.

### ✨ Features
- **8 Artistic Styles**: Vivid Pop, Vintage Sepia, Oil Painting, Neon Edge, Dreamy Glow, Cyberpunk, Monochrome, Watercolor
- **Real-time AI Analysis**: MobileNet identifies objects in your images
- **Smart Enhancements**: AI adjusts colors based on image content
- **Drag & Drop**: Easy image upload
- **Download Results**: Save your AI artwork as PNG

### 🛠️ Tech Stack
- TensorFlow.js
- MobileNet (Image Classification)
- HTML5 Canvas API
- CSS3 Animations
